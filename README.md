# Gitops
Local kubernetes에서 Jenkins와 argo cd 이용한 gitops 구축하고  
  
Prometheus와 Grafana를 이용하여 Monitoring 환경을 구축한다.  

CI CD 테스트 소스 코드는 Fastapi로 진행하였다.
